APIs.json Editor

This is an editor for this repositories APIs.json file, looking for the file in the root, loading it, and making it editable through this interface, and save using the Github API.

For each API, when there is a Swagger file present as a proprety, you can click on the title and edit using the Swagger JSON editor. The goal is to make the APIs.json, and all machine readable properties editable here on the page.
